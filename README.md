# Assignment-Round-Nexgensis-Technologies-Pvt.-Ltd.
This project simulates a one-day delivery operation for a fictional logistics company.
Packages are assigned to the nearest delivery agents, deliveries are simulated, and a performance report is generated.

The solution is implemented using Python and Object-Oriented Programming (OOP).
# Files
assignment.py   # Main program
data.json       # Input data
report.json     # Generated output
